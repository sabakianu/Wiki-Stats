# Wiki-Stats

Wiki-Stats is a project designed to analyze Wikipedia data stored in ZIP archives containing JSON files. The key functionalities of the project include:

- **Reading ZIP Archives**: Efficiently unzips and reads JSON files containing Wikipedia data.
- **Calculating Word Frequencies**: Computes the frequency of words across the Wikipedia articles, allowing users to understand the most common terms used.
- **Finding Longest Articles/Titles**: Identifies the longest articles and titles within the dataset, providing insight into the most extensive content available on Wikipedia.

This project aims to help users analyze and gain insights from large amounts of Wikipedia data easily and efficiently.
